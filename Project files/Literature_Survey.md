# Literature Survey

## Epic 1: Define Problem and Understanding

## Overview

A literature survey helps identify existing research, technologies, and
methodologies related to crop recommendation systems and intelligent
agriculture. It provides the foundation for selecting suitable machine
learning techniques and understanding current challenges in agricultural
decision support systems.

------------------------------------------------------------------------

## Purpose

The primary objective of this survey is to study how Artificial
Intelligence (AI), Machine Learning (ML), and data analytics are applied
in modern agriculture to improve crop selection, productivity, and
resource management.

------------------------------------------------------------------------

## Existing Research Areas

The survey includes studies on:

-   Crop recommendation systems
-   Crop yield prediction
-   Soil nutrient analysis
-   Precision agriculture
-   Smart farming technologies
-   Environmental monitoring
-   Decision support systems

------------------------------------------------------------------------

## Machine Learning Techniques Studied

The following algorithms are commonly used in agricultural prediction
systems:

-   K-Nearest Neighbors (KNN)
-   Logistic Regression
-   Decision Tree
-   Random Forest
-   Support Vector Machine (SVM)
-   Neural Networks
-   K-Means Clustering

Each algorithm offers different advantages in terms of prediction
accuracy, computational efficiency, and suitability for agricultural
datasets.

------------------------------------------------------------------------

## Data Preprocessing Techniques

The literature review also examines common preprocessing methods,
including:

-   Handling missing values
-   Outlier detection
-   Feature engineering
-   Feature scaling
-   Dataset balancing
-   Train-test data splitting

These techniques improve the quality of agricultural datasets and
enhance model performance.

------------------------------------------------------------------------

## Evaluation Metrics

Researchers commonly evaluate crop recommendation models using:

-   Accuracy
-   Precision
-   Recall
-   F1-Score
-   Confusion Matrix
-   Cross Validation

These metrics help compare different machine learning models and
identify the most reliable solution.

------------------------------------------------------------------------

## Research Gaps

The literature highlights several challenges:

-   Limited use of real-time environmental data.
-   Regional variations in soil and climate conditions.
-   Scalability issues with large agricultural datasets.
-   Lack of integrated web-based recommendation platforms.
-   Need for sustainable and resource-efficient farming solutions.

These gaps motivate the development of the OptiCrop system.

------------------------------------------------------------------------

## Proposed Contribution

OptiCrop addresses these limitations by integrating soil nutrients,
climate parameters, and machine learning into a single intelligent crop
recommendation platform. The system aims to deliver accurate
recommendations through a simple web interface while promoting
sustainable farming and efficient resource utilization.

------------------------------------------------------------------------

## Conclusion

The literature survey demonstrates that machine learning has significant
potential to improve agricultural decision-making. By studying existing
approaches and identifying their strengths and limitations, the OptiCrop
project adopts suitable methodologies to build a reliable, scalable, and
intelligent crop recommendation system.
