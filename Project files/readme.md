# Project Files

## Overview

This folder contains all the project documentation, reports, and implementation files created during the development of the **OptiCrop: Smart Agricultural Production Optimization Engine**.

These documents were prepared as part of the project development workflow and include the activities completed in each epic, along with their corresponding tasks and implementation details.
